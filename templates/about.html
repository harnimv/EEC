About Our Web App

Welcome to Our Flask App!

This web application was created using Flask and SQLite to showcase the power of simple web development. It allows users to register and view a list of other users.

Our Mission

Our mission is to provide a basic framework for learning how to build web apps with Python and Flask. We believe that simplicity is key, and this app is designed to be both easy to understand and easy to extend.

Created with love by the Flask community.
